Projetos

Atualmente, estou desenvolvendo um site chamado teknews.com.br,teknews.com.br, utilizando JavaScript como linguagem principal. O projeto é containerizado com Docker, garantindo um ambiente de desenvolvimento isolado e reproduzível. Os testes automatizados são implementados com Jest, assegurando uma cobertura robusta e a confiabilidade do código. Além disso, estou utilizando uma extensão para padronização de mensagens de commit, o que melhora o entendimento do histórico de alterações e facilita práticas como versionamento semântico e geração automática de changelog.
